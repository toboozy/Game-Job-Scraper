SETUP AUTOMATION OF JOBSCRAPER

This .exe will run jobscraper.py, creating if it doesn't exist, or updating if it does, the jobs.db database in the same folder it's run.

Ensure this exe is in the folder you'd like to create the db. 

WINDOWS

	1. Go to Task Scheduler
	2. Create Basic Task
	3. Set Action as Start a Program
	4. Set the program/script to jobscraper.exe
	5. Ensure 'Start in' is the folder jobscraper.exe is in
	6. DONE!
